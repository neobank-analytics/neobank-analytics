# Neobank Analytics

Dashboard interactif d'analyse d'une neobank (Jan 2018 – Mai 2019) — segmentation clients, prédiction du churn et détection de fraude.

**Par Arthur & Matthieu**

---

## Modules

- **Base Clients** — KPIs, inscriptions, appareils, notifications
- **Transactions** — volumes, types, devises, comportements
- **Machine Learning** — Clustering K-Means, Churn Random Forest, Fraude Isolation Forest

## Stack

`Python` `Streamlit` `Scikit-learn` `Plotly` `Pandas`
